打包
===

chrome插件打包方式

1. 打开chrome://extensions/
2. 勾选开发者模式
3. 点击打包扩展程序，选择项目目录
4. 打包完成


安装
=== 

0. 下载本项目到本地，并解压
1. 打开chrome://extensions/
2. 勾选开发者模式
3. 点击加载已解压的扩展程序，选择项目目录
4. 安装完成
